# 架构设计

ProIview 是基于 iview 而开发的模板组件，提供了更高级别的抽象支持，开箱即用。可以显著的提升制作 CRUD 页面的效率，更加专注于页面。

ProTable 表格模板组件，抽象网络请求和表格格式化

ProForm 表单模板组件，预设常见布局和行为

ProSearch 搜索模板组件，预设常见布局和行为
